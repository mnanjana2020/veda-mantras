# Veda Mantras

A website showcasing Veda mantra translations and analysis with authentic Indian cultural aesthetics.

## About

This project presents translations and interpretations of ancient Vedic mantras, designed with traditional Indian artistic elements to honor the sacred nature of these texts.

## Structure

- **Rig Veda** - Hymns of praise and worship
- **Yajur Veda** - Sacrificial formulas and rituals
- **Sama Veda** - Musical chants and melodies
- **Atharva Veda** - Spells, charms, and philosophical hymns
